## Description

Pizza Pi is a modern restaurant management system built with React and Vite, designed to streamline restaurant operations. It features menu management, order processing, table management, and an admin dashboard for restaurant staff.


## Usage

### Customer Features
- Browse the restaurant menu
- Place orders
- Track order status
- Request payment

### Admin Features
- Manage menu items
- Process orders
- Handle payments
- Manage tables
- View analytics

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Supabase
- React Router
- Tanstack Query
